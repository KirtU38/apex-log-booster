Minimal requirements for the log levels:
- Database: Info
- Workflow: Info
- Validation: Info
- Apex Code: Finer
- Apex Profiling: Info

How to Use:
Open any .log file, right click and select "Log: Order of Execution" or "Log: SOQL Analyzer".
